# StockInsight

해외 주식 분석 대시보드

- 프리마켓 / 전일 정규장 등락률 실시간 표시
- 뉴스 한국어 번역 및 AI 요약
- 워치리스트 자동 뉴스 갱신

## 기술 스택

- **Frontend:** Next.js 16, TypeScript, Tailwind CSS v4
- **Backend:** Supabase (PostgreSQL + Auth + Edge Functions)
- **AI:** Claude Haiku (뉴스 번역·분류·시장 분석)
- **실시간 가격:** Finnhub WebSocket
- **주가 데이터:** Financial Modeling Prep (FMP) Stable API
