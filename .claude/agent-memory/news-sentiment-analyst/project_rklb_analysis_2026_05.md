## 섹터 패턴 메모
- RKLB는 수주 계약 공시가 주가에 즉각적이고 강한 호재 반응을 유발하는 패턴
- 대규모 ATM 주식 매도 프로그램은 희석 우려로 단기 악재로 분류되나 자본 조달 목적이므로 중장기 영향은 중립~긍정 가능
- 방산(DoD, Space Force) 계약이 뉴스 호재의 주요 원천

**Why:** 첫 분석 세션에서 RKLB 관련 주요 이슈 패턴을 누적하여 재분석 시 비교 참조 가능하도록 저장
**How to apply:** 다음 RKLB 분석 요청 시 이번 분석 결과와 비교하여 심리 변화 추이 제공
